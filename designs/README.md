# designs/

This folder holds original source files and editable designs intended for laser cutting.

Use this folder for:
- SVG, AI, DXF, PDF, and other source artwork files
- Layered design files with cut/engrave separation
- Editable versions you may refine later

Suggested naming convention:
- `project-name-version.svg`
- `tag-design-ai.ai`
- `clock-face-engrave.dxf`

Example:
- `keychain-v1.svg`
- `wall-art-6x6.ai`
- `jewelry-tag-template.dxf`
