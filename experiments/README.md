# experiments/

This folder is for test cuts, experiments, and trial pieces used to dial in settings or validate new designs.

Use this folder for:
- speed/power tests and cut samples
- proof-of-concept pieces and mockups
- notes on what worked, what failed, and improvements

Suggested naming convention:
- `project-test-description.md`
- `bottle-opener-speed-test.md`
- `engrave-wood-sample.md`
- `acrylic-kerf-check.md`

Example:
- `tag-cut-test-3mm-wood.md`
- `font-engrave-sample.md`
- `material-edge-quality.md`
