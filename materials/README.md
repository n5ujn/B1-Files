# materials/

This folder stores material notes, cut settings, and preset information for the laser cutter.

Use this folder for:
- material tests and power/speed settings
- notes on cut quality, number of passes, and kerf
- material-specific instructions and recommendations

Suggested naming convention:
- `material-thickness.md`
- `acrylic-3mm.md`
- `birch-plywood-5mm.md`
- `leather-2mm.md`

Example:
- `basswood-3mm.md`
- `black-acrylic-4mm.md`
- `chipboard-2mm.md`
