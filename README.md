# Longer B1 36W Laser Cutter Projects

This repository is a home for files you have downloaded or designed for cutting with your Longer B1 36W laser cutter.

## Purpose

Use this repo to store, organize, and track laser cutting designs, test cuts, and material settings. Keep your SVG, DXF, PNG, PDF, and other laser-ready files together with notes on power, speed, and material type.

## Recommended Structure

Organize files in folders by project or material type so they remain easy to find:

- `designs/` – original design files, source artwork, and editable files
- `cuts/` – exported cutting files ready for the cutter
- `materials/` – notes and presets for wood, acrylic, leather, etc.
- `experiments/` – test pieces, settings notes, and proof-of-concept cuts

## How to Use

1. Add your design source files (SVG, DXF, AI, etc.) to the repo.
2. Export cutting-ready files from your design tool and save them with a clear name.
3. Record material, power, speed, and passes in a text file or within the folder name.
4. Commit changes so you can track what worked and what did not.

## File Naming Tips

Use descriptive names and include material or cut type if possible:

- `name-material-thickness.svg`
- `project-name-acrylic-3mm.dxf`
- `tag-engrave-wood-5mm.png`

## Notes

- The Longer B1 36W is best suited for thin wood, acrylic, leather, and paper.
- Keep a log of cut settings for each material and thickness.
- Save any photos or notes with the design so you can reproduce results later.

B1-Files/

## Goal

Keep this repository as your personal laser cutting library and reference for the Longer B1 36W. Over time, it should become a reliable place to store successful cut files, presets, and build notes.
