# cuts/

This folder contains laser-ready files exported from your source designs, ready for the Longer B1 cutter.

Use this folder for:
- final cutting files in the format your software/cutter accepts
- files with cut and engrave paths separated into correct layers
- exported versions of source designs for actual production

Suggested naming convention:
- `project-name-material-thickness.format`
- `keychain-acrylic-3mm.dxf`
- `sign-birch-5mm.svg`
- `label-engrave-wood.pdf`

Example:
- `pendant-wood-3mm.svg`
- `coaster-acrylic-4mm.dxf`
- `nameplate-engrave-5mm.pdf`
